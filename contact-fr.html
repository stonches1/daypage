Foire aux questions

Mes données sont-elles privées ?

Oui ! Toutes vos données sont stockées localement sur votre appareil par défaut. Si vous activez la synchronisation iCloud, vos données sont stockées dans votre compte iCloud personnel avec le chiffrement de bout en bout d'Apple. Nous n'accédons jamais à vos entrées de journal.

Comment restaurer mes achats ?

Ouvrez DayPage, allez dans Réglages, appuyez sur « Passer Premium » et cherchez l'option « Restaurer les achats ». Assurez-vous d'être connecté avec le même identifiant Apple utilisé pour l'achat original.

Comment fonctionne la synchronisation iCloud ?

La synchronisation cloud utilise votre compte iCloud personnel pour garder vos entrées de journal synchronisées sur tous vos appareils Apple. Vous pouvez l'activer ou la désactiver à tout moment dans les Réglages. Vos données restent dans votre compte iCloud et nous n'y avons jamais accès.

Comment fonctionne le Mode Duo ?

Le Mode Duo vous permet de tenir un journal avec un partenaire. Vous choisissez quelles entrées partager, et votre partenaire peut voir ces réflexions partagées. Les deux utilisateurs doivent accepter, et vous pouvez vous déconnecter à tout moment.

Puis-je exporter mes entrées ?

Oui ! Allez dans Réglages > Exporter pour générer un PDF de vos entrées de journal. Le PDF est créé localement sur votre appareil et peut être partagé ou enregistré comme vous le souhaitez.
